How to setup an IPA input schema

2022-11-01
1 min read

Using RIME

There are two schemas, yunlong and xsampa.

A guide to xsampa: https://chromium.googlesource.com/chromiumos/third_party/espeak-ng/+/HEAD/docs/phonemes/xsampa.md

Alternatively, you can simply install Yunlong, which can be accessed here http://118.24.95.172/resource.htm

Typing IPA in \LaTeX

Using the tipa package. A cheatsheet of the symbols can be found in https://ptmartins.info/tex/tipacheatsheet.pdf

Online IPA keyboard

https://keyman.com/keyboards/sil_ipa (SIL keyboard)

https://ipa.typeit.org/full/

http://westonruter.github.io/ipa-chart/keyboard/
