Using `sed` to transform a file in markdown to LaTeX

0001-01-01
1 min read

headings

To transform a heading in MD to a heading in ltx, it is OK to list all six possible headings in MD:

s/^# \(.*\)/\\part{\1}/
s/^## \(.*\)/\\section{\1}/
s/^### \(.*\)/\\subsection{\1}/
s/^#### \(.*\)/\\subsubsection{\1}/
