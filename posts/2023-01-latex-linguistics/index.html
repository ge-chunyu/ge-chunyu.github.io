LaTeX packages for linguistics

2023-01-31
1 min read

As I’ve been using LaTeX to typeset linguistic papers and handouts for quite some time, it’s perhaps time to write some notes on it.

There have been plenty of guide on LaTeX and linguistics, and below lists only a few.

  • WikiBooks on LaTeX and Linguistics
  • Anke Himmelreich’s guide comprehensive
  • LaTeX for Linguists

Trees

There are several packages for drawing trees. In addition to tikz and qtree, there is also the forest package. The main difference between qtree and forest is a dot is needed for a node in qtree but not in forest.

Numbered examples

gb4e and ling-macros can number examples, and another option is linguex. The syntax is straightforward. The plain \label and \ref functions can be used to reference the example.

\ex. \label{example} This is an example.

IPA

I usually input IPA using a IPA input method, as said in the previous post. But tipa is still useful, when xelatex is not possible when using journal and conference templates.

Another useful package is vowel, which can draw vowel charts, specify its shape, and choose which vowels to include in the chart. The vowel chart can be a triangle or rectangle. \putcvowel can be used to put cardinal vowels to the chart, and \putvowel is used to put vowel at the position specified by the coordinates determined by the following two parameters. The first option specifies the position of the vowel, whether its to the left or right of the chart.

\begin{vowel}[plain,triangle,rectangle]
\putcvowel[l]{i}{1}
\putcvowel[r]{u}{8}
\putvowel[r]{a}{2\vowelhunit}{\vowelvunit}
\end{vowel}

Phonological rules

The phonrule package does a good job in typesetting phonological rules. The usage is also simple and straightforward.

\phon{A}{B} # A changes to B
\phonl{A}{B}{C} # A changes to B after C
\phonr{A}{B}{C} # A changes to B before C
\phonb{A}{B}{C}{D} # A changes to B between C and D

To write rules using feature matrix, use \phonfeat{} and include the features in the curly brackets.
